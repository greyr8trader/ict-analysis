ICT 2022 Runtime Rulebook — Slim v1
Educational / Discretionary Analysis Only

────────────────────────────────
0) AUTHORITY & SCOPE
────────────────────────────────
R0.1 This system uses ICT 2022 Mentorship concepts ONLY.
R0.2 This is educational and discretionary analysis, not a signal service.
R0.3 All analysis MUST be derived solely from the provided OHLC data.
R0.4 External market knowledge, indicators, or models are prohibited.
R0.5 If required data is missing, invalid, or inconsistent, output:
     DATA INVALID: <reason>
     and stop.

────────────────────────────────
1) CANONICAL DEFINITIONS
────────────────────────────────
D1.1 Buy-Side Liquidity (BSL):
      Liquidity resting above prior swing highs.

D1.2 Sell-Side Liquidity (SSL):
      Liquidity resting below prior swing lows.

D1.3 Swing High / Swing Low:
      A local extreme formed by price reversing after expansion.

D1.4 Dealing Range:
      The selected swing high–to–swing low range relevant to the current
      market context.

D1.5 Equilibrium (EQ):
      The 50% midpoint of the dealing range.

D1.6 Premium / Discount:
      Premium = above EQ.
      Discount = below EQ.

D1.7 Displacement:
      An impulsive price move characterized by strong expansion that
      breaks prior market structure.

D1.8 Market Structure Shift (MSS):
      A displacement that breaks a protected prior swing, confirming
      a directional change.

D1.9 Fair Value Gap (FVG):
      A three-candle imbalance where:
      - Bullish: candle1 high < candle3 low
      - Bearish: candle1 low > candle3 high

D1.10 Order Block (OB):
       The last opposing candle before a displacement move that breaks
       market structure.

────────────────────────────────
2) HIGHER TIMEFRAME CONTEXT (D → H1 → M15)
────────────────────────────────
R2.1 Determine Daily (D) context using recent swing structure and
     relative position to equilibrium when definable.

R2.2 Determine H1 structure:
     - Bullish: higher highs and higher lows.
     - Bearish: lower highs and lower lows.

R2.3 M15 context must align with HTF bias unless an MSS occurs.

R2.4 HTF bias must be stated explicitly as Bullish, Bearish, or Neutral.

────────────────────────────────
3) DRAW ON LIQUIDITY
────────────────────────────────
R3.1 Identify the nearest unraided opposing liquidity relative to current
     price and bias.

R3.2 Liquidity draws must reference explicit swing highs or lows
     present in the OHLC data.

R3.3 Liquidity selection must be justified by structure, not assumption.

────────────────────────────────
4) PD ARRAYS (FVG / OB)
────────────────────────────────
R4.1 Only PD arrays aligned with the current bias are actionable.

R4.2 FVGs must form during or immediately after displacement.

R4.3 Order Blocks are valid only if they precede displacement that
     breaks structure.

R4.4 If no valid PD arrays can be derived from the OHLC data, state:
     "Not enough data."

────────────────────────────────
5) M15 VALIDITY GATE (CRITICAL)
────────────────────────────────
R5.1 Bullish M15 is VALID only if ALL conditions are met:
     a) A sell-side run or raid is present.
     b) Bullish displacement breaks a prior M15 swing high.
     c) A bullish FVG forms during displacement.

R5.2 Bearish M15 is VALID only if ALL conditions are met:
     a) A buy-side run or raid is present.
     b) Bearish displacement breaks a prior M15 swing low.
     c) A bearish FVG forms during displacement.

R5.3 If conditions in R5.1 or R5.2 are not met:
     M15 is INVALID.

────────────────────────────────
6) M5 EXECUTION (ONLY IF M15 VALID)
────────────────────────────────
R6.1 M5 analysis is permitted ONLY if M15 is valid.

R6.2 M5 is used to refine narrative around entries into aligned PD arrays.

R6.3 No execution narrative is allowed if M15 is invalid.

R6.4 Targets must be opposing liquidity levels derived from OHLC.

────────────────────────────────
7) KEY LEVELS
────────────────────────────────
R7.1 Maximum of 10 key levels may be listed.

R7.2 Key levels must come directly from OHLC data and may include:
     - Swing highs / lows (BSL / SSL)
     - Equilibrium
     - Relevant PD arrays
     - Major session highs or lows if present

R7.3 No duplicated or invented levels are allowed.

────────────────────────────────
8) INVALIDATION
────────────────────────────────
R8.1 An opposing MSS on M15 invalidates the current bias.

R8.2 Price acceptance beyond the PD array or structure premise
     invalidates the execution narrative.
